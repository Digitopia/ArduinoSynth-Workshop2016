Descrição

Criar um sintetizador ou um processador de efeitos nunca foi tão fácil. Com a evolução da plataforma de prototipagem electrónica Arduino, disponível em open source, e a contribuição de bibliotecas dedicadas à geração de sinal áudio, torna-se cada vez mais apetecível a criação de synths/effects pedals costumizados. É sobre esta liberdade criativa, acessível a todos, que incide o workshop.

Formador

Tiago Ângelo

Plano

Sábado 23.04.2016

Desenvolvimento de um processador de efeitos: começaremos por usar um piezo como microfone de contacto e passaremos para a construção de um circuito para entrada de linha.

Domingo 24.04.2016

Desenvolvimento de um sintetizador: enviando MIDI para o sintetizador via USB. (Opcionalmente será descrito como criar um circuito para receber MIDI via cabo DIN)

NOTAS

Este workshop é focado no código e na construção de circuitos simples, ficando a cargo do participante a posterior montagem do sintetizador ou processador de efeitos numa caixa.

Lista de Material

Placas recomendadas (Arduino)
  • Mega (RECOMENDADO)
  • Uno (RECOMENDADO)
  • Leonardo
  • Duemilanove
  • Nano
  • Pro Mini
  • *Outras placas: Ardweeny, Borduino, Freetronics EtherMega, Teensy2, Teensy2++, Teensy3.1

    • *NOTA: o workshop será centrado em Arduino pelo que para usar as outras placas os utilizadores deverão estar à vontade com a sua utilização.

*NOTA: para domingo iremos enviar MIDI para o Arduino por USB através do computador, no entanto se quiserem montar o circuito para ligar um cabo MIDI ao Arduino deverão trazer o material descrito para domingo, para além de um dispositivo capaz de enviar MIDI para o Arduino por cabo. Recomenda-se também que tragam computador para programar o Arduino, embora haja 5 disponíveis na digitópia. Para ouvir o som do vosso sintetizador ou processador de efeitos poderão ligá-los ao vosso computador (com cabo minijack) ou trazer pequenos altifalantes amplificados. Também é possível usar auscultadores embora se devam usar com precaução para não danificar os ouvidos enquanto se testa o código!
